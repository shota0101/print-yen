# 硬貨学習用A4印刷ページ作成ツール

子どもの金銭感覚を育てるための、硬貨学習用A4印刷ページを簡単に作成できるツールです。

## 概要

このツールは以下の機能を提供します：

- **カスタムページ作成**：値段と硬貨の組み合わせを選択して、オリジナルの学習ページを生成
- **実物サイズ対応**：硬貨を実際のサイズで印刷
- **複数硬貨対応**：1円、5円、10円、50円、100円、500円に対応
- **グループ単位での改行**：「硬貨を追加」ボタンを押すたびに改行（ユーザーの意図通りに配置）
- **A4印刷最適化**：point.md のベストプラクティスに基づいた印刷設定

## 使い方

### 1. ページを開く

- **本番環境**：`index.html` をブラウザで開く（またはGitHub Pagesでホスト）
- **サンプル確認**：`sample.html` で既製テンプレートを確認

### 2. 値段と硬貨を設定

1. **「値段を追加」** をクリックして金額を入力
2. **「硬貨を追加」** をクリックして以下を入力：
   - 硬貨の種類（ドロップダウン選択）
   - 枚数
   - **注**：「硬貨を追加」で追加した行ごとに改行されます
3. 複数行設定可能（例：50円玉1枚 + 10円玉5枚）

**例：** 
- 「硬貨を追加」→ 10円玉×5 → 1行目に表示
- 「硬貨を追加」→ 10円玉×5 → 2行目に表示

### 3. 印刷ページを生成

**「表示」** ボタンをクリックすると、新しいウィンドウでA4印刷用ページが生成されます

### 4. 印刷

- ブラウザの印刷機能（Cmd+P / Ctrl+P）で印刷
- または PDF として保存

## 硬貨サイズ

実際の日本硬貨のサイズに合わせて設定：

| 硬貨 | サイズ |
|------|--------|
| 1円  | 20.0mm |
| 5円  | 22.0mm |
| 10円 | 22.6mm |
| 50円 | 20.7mm |
| 100円| 22.6mm |
| 500円| 26.5mm |

## 印刷設定

- **ページサイズ**：A4（210mm × 297mm）
- **余白**：12mm
- **本文幅**：186mm（210mm - 24mm）
- **1行の配置**：
  - 「硬貨を追加」で追加した硬貨のグループごとに改行
  - 同じ硬貨のグループは最大6個まで1行に配置
  - 例）10円玉×5枚 → 1行、10円玉×5枚 → 別の行
